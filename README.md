# INSTALLATION: 

## Run:
```
git clone https://github.com/chegmarco1989/brainimmo.git braintechimmo
```

## Run:
```
cd braintechimmo
```

## Run:
```
composer install
```

## Run:
```
npm install
```

## Run:
```
cp .env.example .env
```

## Run:
```
php artisan key:generate
```

## Run:
```
php artisan migrate
```

## Run:
```
php artisan serve
```

## Go to:
```
http://127.0.0.1:8000
```
