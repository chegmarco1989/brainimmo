## 1. Clone project and install:


## 2. Run:
```
composer install
```

## 3. Run:
```

```


## 4. Run:
```
php artisan key:generate
```

## 5. Run:
```
php artisan migrate
```

## 6. Run:
```
php artisan serve
```
